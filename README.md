# Vladika Andrej V3 Skeleton

This is the "Modern Orthodox" skeleton for the Eparchie of Vladika Andrej.

## Tech Stack
- **Framework:** Next.js (Pages Router)
- **Styling:** Tailwind CSS + Framer Motion
- **Language:** i18next (Multi-language support: DE, SR, IT)
- **Icons:** Lucide React

## Structure
- `src/pages/`: Core sections (Home, News, Bishop, Liturgy, Contact)
- `src/locales/`: i18n JSON files
- `src/components/`: Shared UI components (Navbar, Footer)
- `src/styles/`: Tailwind globals and orthodox theme colors

## Multi-Language
Supports:
1. German (DE)
2. Serbian (Cyrillic + Latin)
3. Italian (IT)

## Quick Start
```bash
npm install
npm run dev
```
